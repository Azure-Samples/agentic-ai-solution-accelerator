# Partner walkthrough

The accelerator is delivered as a **linear walkthrough** in two tracks:

- **[Get ready](ready/01-get-oriented.md)** — three steps you do **once** when you adopt the accelerator (orient, install tools, rehearse in your own sandbox).
- **[Deliver to a customer](deliver/01-clone-for-the-customer.md)** — seven steps you repeat **per engagement** (clone, discover, scaffold, provision, iterate, hand over, operate).

Every page ends with a single **Continue →** link to the next step. Roles are skim guidance, not separate paths:

- **Delivery lead** — read *Get oriented* and *Discover with the customer* deeply; skim the engineering steps; read *UAT & handover* and *Operate (Day 2)*.
- **Partner engineer** — skim *Get oriented*; read *Set up your machine* and *Rehearse in a sandbox* once; then own *Clone* through *Iterate & evaluate*.
- **Solo partner** — read end-to-end.

## Joining mid-engagement?

Jump in at the step that matches your current state.

| If this is true… | Start at |
|---|---|
| You haven't cloned the customer repo yet | [Clone for the customer](deliver/01-clone-for-the-customer.md) |
| Repo is cloned, no brief yet | [Discover with the customer](deliver/02-discover-with-the-customer.md) |
| Brief is filled in the repo, no scaffold yet | [Scaffold from the brief](deliver/03-scaffold-from-the-brief.md) |
| Code scaffolded, not provisioned in customer Azure | [Provision the customer's Azure](deliver/04-provision-the-customers-azure.md) |
| Provisioned, you're customising and running evals | [Iterate & evaluate](deliver/05-iterate-and-evaluate.md) |
| Evals green, heading into UAT | [UAT & handover](deliver/06-uat-and-handover.md) |
| Already in production | [Operate (Day 2)](deliver/07-operate-day-2.md) |

Reference material — Architecture, Responsible AI, agent specs, tool catalog, chatmodes, frontend starter, alternate scenarios — sits **outside the walkthrough** under [Reference](../reference.md). Open it when a step in the walkthrough sends you there.
