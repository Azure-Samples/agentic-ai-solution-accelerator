# Starter Pack

> Minimal example used by Phase 2 60-minute quickstart.
> Single agent, 1 grounding source, no side-effect tools.

## Purpose
Demonstrate in ~60 minutes:
- `baseline materialize all` round-trip.
- Kill switch triggered by an eval probe.
- Cost ceiling hit visibly.
- HITL rejection (using a fake side-effect tool in sandbox only).

## Sandbox-only
This pack targets the `sandbox-only` bundle. Not a production blueprint. See `docs/partner-playbook.md` Phase 2.
